Home Page

This is the home page, at the root of the website. It is named index.html so that it can be automatically loaded by the webserver. As the site is loaded from this page, then all other locations would recognise this to be the top-most or root of the site.

This page also serves as a template to add the other pages. In this example, the other pages are loaded from the docs folder while the project page(s) are loaded from the prj folder

This site is a modification of Jake Wright's Learn CSS in 12 minutes. You should use this as an example and modify your site to match modern pcs and browsers.
